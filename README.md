<h2>📖 Project Story</h2>

<p>
Restaurant businesses need to understand customer preferences to improve their services and attract more customers.
This project analyzes Zomato restaurant data using Python, Pandas, and Matplotlib to identify trends in restaurant ratings,
cuisine popularity, customer engagement, and restaurant performance.
The dashboard transforms raw data into meaningful insights that support data-driven business decisions.
</p>

<hr>

<h2>🎯 Business Problem</h2>

<ul>
    <li>Which restaurants are the most popular?</li>
    <li>Which restaurants receive the highest ratings?</li>
    <li>Which cuisines are preferred by customers?</li>
    <li>Do highly rated restaurants receive more customer votes?</li>
    <li>How are restaurant ratings distributed?</li>
</ul>
